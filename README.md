# Sistema de Gestión de Consolas de Videojuegos
## Descripción
Sistema de datos de consolas, que permite administrar un catalogo
## Funcionalidades
- Agregar consola
- Buscar consola por sigla
- Eliminar consola
- Mostrar todas las consolas
## Cómo ejecutar
Para correr el programa, abre la terminal en la raíz del proyecto y ejecuta:
python consolas.py
